## CRUD echo (Golang) And Mongo

Repository for learning to create API with Golang (echo) And Mongo DB

## Run Locally

Clone the project

```bash
  git clone https://github.com/asdamwongmantap/api-echo-mongo.git
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  go run main.go
```
